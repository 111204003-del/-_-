# 🚇 Taipei MRT Map – 台北捷運雙語互動地圖

一個完全在前端運行的台北捷運路線圖查詢工具，支援**中英文即時切換**、**最短路徑計算**、**票價試算**及**乘車時間預估**。無需後端、無需 API，點開即用。

🔗 **線上體驗**：將本倉庫部署到 GitHub Pages 即可使用  
📱 **響應式設計**：電腦、平板、手機皆可操作

## ✨ 功能亮點

- 🗺️ **完整捷運路網**：板南線、淡水信義線、松山新店線、中和新蘆線、文湖線、環狀線（含蘆洲支線）
- 🖱️ **直覺操作**：左鍵站點設為起點，右鍵設為終點
- 🌐 **中英文雙語**：點擊右上角按鈕，所有站名、按鈕、票價資訊立即切換
- 🧮 **智慧路徑**：基於 Dijkstra 演算法計算實際軌道距離
- 💰 **票價試算**：依台北捷運累進費率（5公里內20元，每3公里加5元，上限65元）
- ⏱️ **乘車時間**：依距離與站間停等估算
- 🔄 **交換起訖站**：一鍵互換起點與終點
- 🎨 **視覺標示**：起點藍色、終點紅色、懸停橘色

## 🚀 部署方式

### 1. 上傳到 GitHub Pages

1. 將本倉庫 clone 或下載
2. 確保 `index.html` 位於根目錄
3. 進入 GitHub 倉庫 Settings → Pages → 選擇 `main` 分支 / root 資料夾
4. 存取產生的網址即可

### 2. 本地使用

直接下載 `index.html`，雙擊用瀏覽器開啟（建議 Chrome / Edge / Safari）。

## 🛠️ 技術細節

- 純原生 HTML/CSS/JS，無任何第三方依賴
- SVG 動態繪製路線及站點
- 圖論最短路徑 (Dijkstra) 完全在前端計算
- 響應式布局，支援觸控與右鍵功能

## 📌 注意事項

- 票價與時間為估算值，實際以台北捷運公司公告為準
- 站點座標為示意相對位置，非真實地理經緯度
